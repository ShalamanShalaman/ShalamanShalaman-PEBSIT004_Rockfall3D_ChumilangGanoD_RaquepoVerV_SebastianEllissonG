# **How to Run the Game**

## **Option 1: Using the Unity Editor**

1. Install Unity Hub and add the Rockfall project folder.

2. Open the main scene from Assets/Scenes/ (e.g., MainScene.unity).

3. Click the Play button in the Editor.


## **Option 2: As a Standalone Application**

1. In Unity, go to File > Build Settings.

2. Select your platform (Windows, Mac, or Linux) and click Add Open Scenes.

3. Click Build and Run, then select a folder to create the executable.
